# word_bank

Word Bank (워드 뱅크) — 수험생 맞춤형 AI 단어장
Word Bank — AI vocabulary bank for exam-focused learners

## v1 범위 (Initial Release Scope)

### 핵심 흐름 (Core Flow)
- 단어 저장: 자동 뜻 + 예문 1개 생성
- 정리: 태그 기반 분류, 필터
- 학습: 기본 플래시카드(단어 ↔ 뜻)

### 프리미엄 가치 (Premium Value)
- 저장 한도 해제(무제한 저장)
- 문맥 맞춤형 해석
- PDF 출력(단어/뜻/예문 포함)

### 과금/전환 (Monetization)
- 저장 한도 도달 시 업그레이드 안내
- 인앱 결제 + 구매 복원

## 비고 (Notes)
- OCR, 스마트 복습 알림, AI 문제 생성은 v1 범위에서 제외

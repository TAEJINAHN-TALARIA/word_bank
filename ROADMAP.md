# ROADMAP

## v1 (초기 버전) — T1~T9 only

### 핵심 흐름 (Core Flow)
- 단어 저장: 자동 뜻 + 예문 1개
- 정리: 태그 기반 분류, 필터
- 학습: 기본 플래시카드

### 프리미엄 가치 (Premium Value)
- 저장 한도 해제(무제한 저장)
- 문맥 맞춤형 해석
- PDF 출력

### 과금/전환 (Monetization)
- 저장 한도 도달 시 업그레이드 안내
- 인앱 결제 + 구매 복원

## Deferred (Post‑v1)
- 스마트 복습 알림 (spaced repetition)
- OCR 스캔
- AI 문제 생성 (독해 지문, 빈칸 문제)
